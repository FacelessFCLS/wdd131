Welcome to the Desert Dungeon!

Dice Roller

The main function of this page was made in JavaScript. A single die can be rolled by clicking the "Roll Dice" button. The page will store up to 10 previous results.

Monsters

On the Monsters page, you can view a handful of monster options. By clicking on the "Beast" or "Undead" links, you can filter by these monster types. Additionally, you can filter by weaker or stronger monsters. Some Monster information is available above the corresponding picture.

Game

In my free time, I made a simple dungeon crawler game. I was excited to use this game in this project but realized too late that I could not fit a .py file into my .html file and have the game be playable. Instead, there is a video of the game. For this page, I have JavaScript in both a .js file and in the .html file just so I could play around with variety.
